Case Study: Spectral Overlay Tool

Purpose

Biosearch Technologies approached me to redesign and redevelop their spectral overlay tool in 2014. During this time Adobe Flash had already begun its decline in favor of HTML5 APIs, and the company recognized that the eventual drop in support for Flash would mean they would need to port the application to use HTML5 APIs.

They also had plans to launch the qPCR Design Lab, a microsite that offered various resources and tools about quantitative polymerase chain reaction (qPCR) targeted to both new and experienced users. The spectral overlay tool would be one of the main tools to be featured on the site.

Role & Responsibilities

I worked as a web developer consultant, responsible for the design of the user interface and flow as well as the programming of the application. Biosearch Technologies’s marketing team would define the high-level feature requirements, and the research team would give me the data used for the data visualizations. Once I created a build, I would need to coordinate with The Spider, a design agency Biosearch Technologies hired for creating the qPCR Design Lab. They would then deploy the application to the site.

Design

The first design was wireframed and wireflowed by both me and the marketing team. This design supported both desktop and mobile devices. We then quickly iterated through several live builds of the application until we settled on the finalized layout.

In 2015, Biosearch Technologies was acquired by LGC. Because of this they reapproached me to perform some design changes that followed their new branding guidelines.

In 2019 the company reapproached me again, as they wanted to redesign both the user interface and flow of the application. This time the design process was handled by Column Five, and due to time constraints the design would only support desktop, pushing mobile device support for later.

Development

User Interface

Because of my previous experience with Bootstrap and jQuery, I decided to use them as I was most comfortable with them. Initially I designed the layout using Bootstrap’s grid system to make iterating through the initial layout tests as fast as possible. Later on we realized it was important to maximize the amount of space the data visualizations took, especially on mobile devices. This wouldn’t work with the default dimensions defined by Bootstrap’s grids, so in order to have finer control I had to replace the grid system code with a custom layout built with CSS media queries.

After looking at a few different data visualization libraries, I ended up choosing Google Charts because of its support for area charts, simplicity, clear documentation, and popularity.

In 2019, we considered porting the whole application over to a new library like React or Vue.js. But since the application was still working fine and we had a tighter schedule, I decided to try to mostly maintain the technology stack and just update Bootstrap from version 3 to 4 as well as update jQuery and Google Charts to their latest versions. Some performance updates in chart drawing and data handling were done during this time as well.

Data Management

Biosearch Technology’s research team provided me with a Microsoft Excel spreadsheet of data. In order to use this data with the application, I wrote a command line data conversion tool in Python to convert the spreadsheet into JSON. In 2019, I ported this conversion tool to JavaScript and Node.js.

Tooling

As I wanted to try out some new things in my workflow, I took the opportunity to use Yeoman for the first time. This introduced me to Grunt which increased my productivity significantly by handling the development server, handling auto-refresh in response to codebase updates, compiling Sass to CSS, linting my JavaScript with JSHint, running unit tests, and handling production builds.

In 2019, I replaced the Yeoman setup in favor of my own custom webpack configuration.

Release & Reception

The spectral overlay tool first went live alongside qPCR Design Lab on October 23, 2014. The microsite was rewarded Best of Web in Genetic Engineering News, vol. 35, no. 4.

Later the qPCR Design Lab was shut down, but the tool was moved to the main Biosearch Technologies website. You can view the application here: https://www.biosearchtech.com/qpcr-multiplex-spectral-overlay-tool
